How to use?








This is a web that, basically, will help you to save some short things that you decide to save. It will save them in the LocalStorage of you Web Browser, so be carefull, if you delete the data (record, cookies, etc) you might lose what you saved here.


To add a thought, just write in the textbox what you want to save, and, when you're done, click the button "Add".
